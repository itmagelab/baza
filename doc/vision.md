# Vision: Консольная программа для хранения паролей

## Описание видения

Baza — это минималистичный и быстрый CLI-инструмент для управления паролями, ориентированный на безопасность, простоту и локальное хранение данных пользователя.

### Ключевые особенности

- Чистый и логичный интерфейс командной строки
- Максимальная автономность: нет зависимости от облака и сторонних серверов
- Открытый, прозрачный код и формат хранения
- Шифрование всех чувствительных данных
- Конфигурируемость через читабельный конфиг-файл

---

## Пример интерфейса (CLI)

Добавить новый пароль:
```
baza -a some::new::login
```

Показать ранее сохраненный пароль:
```
baza -p some::new::login
```

(В дальнейшем могут быть реализованы и другие флаги: удаление, обновление, просмотр всех, экспорт и т.д.)

---

## Хранение конфигурации

Конфигурационный файл хранится по адресу:
```
~/.config/baza/baza.toml
```
База зашифрованных паролей хранится в:
```
~/.baza/data/<storage>/
```
где `<storage>` — это название используемого бекенда хранения. На данный момент поддерживается только `redb`:
- `~/.baza/data/redb/` — путь для хранения файлов базы при использовании бекенда redb

---

## Формат конфигурации

Файл конфигурации называется `baza.toml`, расположен в `~/.config/baza/` и использует формат TOML. Пример:

```toml
[main]
datadir = "/Users/wilful/.baza"
box_delimiter = "::"
bundle_delimiter = ","

[gitfs]

[storage]
type = "Redb"
```
